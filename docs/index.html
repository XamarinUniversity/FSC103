Before you begin

To complete the coding exercise for this class, you will need an F# environment to work with. The labs are all done in the F# REPL, so you can use Windows or macOS. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

In this class, we will explore some of the common data structures and techniques used in F# programming. We will look at utilizing arrays, lists, sequences, records, tuples and discriminated unions to hold and process data.

Lecture

FSC103 - Data Structures.

Objectives

  1. Utilize arrays and lists
  2. Organize and transform data using data structures

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Working with arrays and lists in the REPL
    Exercise
    In this exercise, we will explore both arrays and lists in F# and some of the common operations you use on them.
  2. Create a sequence of triangular numbers
    Exercise
    In this exercise, we will use tuples and sequences to create a function that returns a triangular sequence of numbers.
  3. Create a record type using tuples
    Exercise
    In this exercise, we will create a record type using tuples in which we experiment with the fst and snd keywords.
  4. Convert C# code into a discriminated union
    Exercise
    In this exercise, we will take an existing C# class and turn it into an F# discriminated union.
