Lab goals

The primary goal of this lab is to create a record type using tuples in which we move the starting point using the fst and snd keywords.

  1. Create a record type which describes a tuple that contain two floats
  2. Move the starting point of the tuple.
  3. Create an instance of the record which adds the starting point to the adjustments.
  4. Create a second instance of the record which subtracts the starting point to the adjustments.

Steps

Create a record type using tuples and functions

  1. Create a new F# library project and open the script.fsx file.
  2. Write a record type that describes the following:
    • StartingPoint: a tuple that contains two floats
    • Move: a function that takes in a starting point and a tuple by which to move the starting point. It should return a result of an adjusted starting point of the tuple

Show Code

type PointTransformations = {
    StartingPoint : float * float
    Move : float * float -> float * float -> float * float
}

  1. Create an instance of this record where Move adds the starting point to the adjustments.

Show Code 

let Adding = {
    StartingPoint = (1., 2.) 
    Move = (fun startingpoint displacement ->  
                (fst startingpoint + fst displacement, snd startingpoint + snd displacement))
}

  1. Create a second instance of this instance by copying the first, and modifying the function so that it subtracts the adjustments from the starting point.

Show Code 

let Subtracting = { Adding with Move = (fun startingpoint displacement ->  
                                        (fst startingpoint - fst displacement, snd startingpoint - snd displacement))
}

Summary

During this lab you have worked to create a record type using tuples, you then adjusted the starting point of that tuple and then wrote the functions to move the starting point and add and subtract based on adjustments made.
