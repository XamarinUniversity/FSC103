Duration

10 minutes

Lab goals

The primary goal of this lab is to build upon our knowledge of option types and to gain a deeper understanding of how to use tuples and sequence operations when writing F# code. We will be writing a function that creates triangular sequence of numbers.

  1. Write a function using a tuple holding the current state of a sequence and a counter.
  2. Use this function to create a sequence.
  3. Experiment with different sequence operations

Steps

Create a triangular sequence of numbers

Our goal is to build a successful sequence of triangular numbers

  1. Create a new F# library project and open the script.fsx file.
  2. Write a function that takes a tuple holding the current state of the sequence, and a counter containing the next number to add. Hint: This function should return an option type.

Show Code 

let generateTriangleNumbers (state, counter) = //#1
    Some(state, (state + counter+1, counter+1))

  1. Use this function to create a sequence, don't forget to add a starting value.

Show Code 

let triangulars = Seq.unfold generateTriangleNumbers (1,1)

  1. Select the code and press ctrl > return to test it in the REPL. Experiment with different sequence operations to verify your code.

Show Code 

triangulars //#3
    |> Seq.take 10
    |> Seq.toList

Summary

During this lab you have built upon your knowledge of option types to create a sequence of triangular numbers
